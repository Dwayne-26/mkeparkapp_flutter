# CitySmart App Wireframes

## Screen Specifications

### 1. Welcome Screen
- **Purpose**: First impression and app introduction
- **Key Elements**: 
  - CitySmart logo and branding
  - Welcome message with Milwaukee imagery
  - "Get Started" call-to-action button
  - Quick feature highlights
- **Navigation**: Leads to onboarding flow
- **File**: `welcome_screen_wireframe.png`

### 2. Landing Screen (Dashboard)
- **Purpose**: Main hub for all parking-related activities
- **Key Elements**:
  - Current location display
  - Quick parking search
  - Recent parking spots
  - Active permits status
  - Street sweeping alerts
  - Navigation shortcuts
- **File**: `landing_screen_wireframe.png`

### 3. Parking Search & Map
- **Purpose**: Find and navigate to available parking
- **Key Elements**:
  - Interactive map with real-time availability
  - Filter options (price, distance, type)
  - Spot reservation functionality
  - Walking directions
  - Parking rates display
- **File**: `parking_map_wireframe.png`

### 4. Permit Management
- **Purpose**: Digital parking permit handling
- **Key Elements**:
  - Active permits list
  - Renewal notifications
  - Payment integration
  - QR code display for verification
  - Permit history
- **File**: `permit_screen_wireframe.png`

### 5. Street Sweeping Notifications
- **Purpose**: Avoid parking violations
- **Key Elements**:
  - Schedule calendar view
  - Location-based alerts
  - Move car reminders
  - Historical violation data
- **File**: `street_sweeping_wireframe.png`

### 6. Parking History
- **Purpose**: Track parking usage and expenses
- **Key Elements**:
  - Chronological parking list
  - Cost analytics
  - Favorite spots
  - Export options
- **File**: `history_screen_wireframe.png`

## Design Notes

### Responsive Design
- All wireframes support both portrait and landscape orientations
- Adaptive layouts for various screen sizes (phones and tablets)
- Accessibility considerations for screen readers

### Color Coding
- Available spots: Green
- Limited availability: Orange
- Occupied/Unavailable: Red
- Reserved: Blue
- Milwaukee branding: Navy blue and cream

### Typography
- Headers: Roboto Bold
- Body text: Roboto Regular
- Accessibility: Minimum 16px font size

## Interactive Elements

### Buttons
- Primary actions: Filled buttons with rounded corners
- Secondary actions: Outlined buttons
- Destructive actions: Red color scheme

### Maps
- Custom markers for different parking types
- Clustering for high-density areas
- Smooth zoom and pan interactions

### Forms
- Clear validation messages
- Progressive disclosure for complex inputs
- Auto-completion for addresses

## Implementation Priority

1. **Phase 1**: Welcome, Landing, Basic Parking Search
2. **Phase 2**: Permit Management, Payment Integration
3. **Phase 3**: Street Sweeping, History, Analytics
